File Handling:
Reading and writing data to files using java classes
like FileInputStream, FileOutputStream, Properties and XSSFWorkBook.
FileInputStream is used to read the data from a file and
FileOutputStream is  used to write the data to the file.
Notes.
Before we write the data to any file we should always load the data
from that file first otherwise it will be overwritten.

==========================================================================================

Работа с файлами:
Чтение и запись данных в файлы с использованием классов java
таких как FileInputStream, FileOutputStream, Properties и XSSFWorkBook.
FileInputStream используется для чтения данных из файла и
FileOutputStream используется для записи данных в файл.
Примечания.
Перед записью данных в любой файл мы всегда должны сначала загрузить данные
из этого файла, иначе они будут перезаписаны.
